## Introduction
This project aims to analyze the data set of "Recipies and Ratings".
This dataset has 17 features and 234429 examples. The features have been listed below:
1. name
2. id
3. minutes
4. contributor_id
5. submitted
6. tags
7. nutrition
8. n_steps
9. steps
10. description
11. ingredients
12. n_ingredients
13. user_id
14. recipe_id
15. data
16. rating
17. review
